# 发布文档 (release/)

存放发布相关文档，包括验收报告和回滚手册。

## 目录职责

本目录存放发布检查清单、验收报告、回滚手册等与发布相关的正式文档。这些文档记录发布决策和执行结果。

## 文件清单

| 文件 | 地位 | 功能 |
|------|------|------|
| `ACCEPTANCE-2026-05-05.md` | 验收报告 | 2026-05-05 版本验收报告 |
| `ROLLBACK.md` | 回滚手册 | 应用与数据库回滚手册 |

## 更新规则

- 每次重大发布后添加验收报告
- 回滚手册根据实际情况更新
- 保留历史版本以供审计
