# Util Package (工具类包)

> 一旦我所属的文件夹有所变化，请更新我。

## 功能作用

存放通用工具类，提供跨模块的辅助功能。

## 文件清单

| 文件 | 地位 | 功能 |
|------|------|------|
| `PasswordValidator.java` | Util | 密码强度验证工具 |

## 工具类职责

- **无状态**: 只包含静态方法
- **无依赖**: 不依赖业务模块
- **可复用**: 可在任意模块中使用

## 密码验证规则

- 最小长度: 8 字符
- 必须包含: 大写字母、小写字母、数字
- 可选: 特殊字符（增加强度）

## 使用示例

```java
if (!PasswordValidator.validate(password)) {
    throw new ValidationException("密码强度不足");
}
```
